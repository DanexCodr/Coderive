# Standard Library Expansion Ideas

## Core Collections
- Stable list, map, set, queue, stack abstractions.
- Iteration utilities: map/filter/reduce/chunk/groupBy.

## String Utilities
- Unicode-aware slicing and normalization.
- Formatting helpers and robust templating utilities.

## Date/Time
- Time zones, durations, parsing/formatting standards.

## File System API
- Cross-platform file read/write helpers.
- Safe path utilities and directory traversal APIs.

## Networking
- HTTP client primitives with retries and timeout handling.
- URL and query-string helpers.

## JSON/YAML/TOML
- First-class parsing and serialization.
- Schema validation utilities.

## Math
- Statistical functions, matrices, random distributions.
- Deterministic random mode for tests.

## CLI Utilities
- Argument parser, color output, progress bars, tables.

## Testing Helpers
- Assertion library and snapshot testing support.
