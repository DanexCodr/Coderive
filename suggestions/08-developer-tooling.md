# Developer Tooling Roadmap Ideas

## Language Server (LSP)
- Autocomplete, hover docs, diagnostics, go-to-definition.

## Formatter
- Canonical formatting tool with stable output.
- Configurable style profile where needed.

## Linter
- Best-practice rules and anti-pattern detection.
- Project-level lint config support.

## REPL Improvements
- Multi-line editing, history, and command helpers.
- Introspection commands for types and symbols.

## Project Scaffolding
- `coderive new` to bootstrap apps/libraries/tests.

## Debugger
- Breakpoints, variable inspection, stepping.

## Build Integration
- Better Gradle/Maven/npm interop examples for mixed stacks.
