Language Examples

Interactive Demo with Multi-Return Slots

unit sample get {
    cod.Math
}

share InteractiveDemo {
    
    // Multi-return slot declaration
    ~| formula, operation
    local calculate(int a, int b, string op) {  
        if op == "+" {
            ~ formula a + b
            ~ operation "addition"
        } else if op == "-" {
            ~ formula a - b  
            ~ operation "subtraction"
        }
    }
    
    share main() {
        output "=== CODERIVE INTERACTIVE DEMO ==="
        
        // Using multi-return values
        result, operationType = [formula, operation]:calculate(10, 5, "+")
        output "Result:" + result + ", Operation:" + operationType
    }
}

Smart For-Loops with Complex Steps

share numberSeries() {
    output "=== Smart For-Loops ==="
    
    // Multiplicative steps
    for i by *2 in 1 to 32 {
        output "Doubling:" + i  // 1, 2, 4, 8, 16, 32
    }
    
    // Compound assignment steps
    for i by i += 2 in 1 to 10 {
        output "Step +2:" + i  // 1, 3, 5, 7, 9
    }
    
    // Division steps
    for i by /2 in 64 to 2 {
        output "Halving:" + i  // 64, 32, 16, 8, 4, 2
    }
}

Multi-Return Slots

Declare multiple return values with ~| syntax and assign with ~ operator

Smart For-Loops

Support for multiplicative (*2), divisive (/2), and compound (i+=1) steps

Flexible Control Flow

Support for both else-if and elif syntax with clean block structure

Array Support

Native array literals with indexing and modification capabilities

Execution Runners

InterpreterRunner

Execute Coderive code directly with the interpreter for fast development cycles.

  • Immediate code execution
  • Built-in debugging support
  • AST visualization option
java -cp coderive.jar cod.runner.InterpreterRunner program.cod

CompilerRunner

Compile Coderive code to bytecode or native assembly for deployment.

  • Bytecode compilation
  • Native assembly output
  • MTOT optimization
java -cp coderive.jar cod.runner.CompilerRunner --native program.cod

CommandRunner

Unified runner with multiple execution modes in a single command.

  • Interpret or compile
  • Flexible output options
  • All features in one tool
java -cp coderive.jar cod.runner.CommandRunner -c program.cod

CoderiveREPL

Interactive environment for experimenting with Coderive code.

  • Real-time evaluation
  • Multi-line input support
  • State management
java -cp coderive.jar cod.runner.CoderiveREPL

Technical Innovations

Mobile-First Architecture

Built entirely on Android devices using Java NIDE, Quickedit, and Termux, proving serious development can happen outside traditional environments.

Dual Parser System

Both ANTLR-based grammar parsing and custom recursive descent parser for robust language processing and error recovery.

Advanced Register Allocation

Hybrid "future-cost" spilling algorithm that predicts register usage patterns for optimal performance.

Multi-Target Compilation

Generate native assembly for both ARM64 and x86_64 architectures from a single Java codebase.

Getting Started

1. Installation

Clone the repository and build the project:

git clone https://github.com/DanexCodr/Coderive.git cd Coderive ./gradlew build

2. Write Your First Program

Create a simple Coderive program:

output "Hello, Coderive!"

3. Run Your Code

Execute your program with the interpreter:

java -cp build/libs/coderive.jar cod.runner.InterpreterRunner hello.cod
