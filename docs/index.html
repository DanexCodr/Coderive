Code Examples

// Simple variable assignment and output
name = "Coderive"
version = 1.0
output "Welcome to " + name + " v" + version

// Conditional logic
x = 10
if x > 5 {
    output "x is greater than 5"
} else {
    output "x is 5 or less"
}

// Looping constructs
for i in 1 to 5 {
    output "Iteration: " + i
}

// Using slots for data storage
~ result 42
output "The answer is: " + ~result

Execution Runners

InterpreterRunner

Execute Coderive code directly with the interpreter for fast development cycles.

  • Immediate code execution
  • Built-in debugging support
  • AST visualization option
java -cp coderive.jar cod.runner.InterpreterRunner program.cod

CompilerRunner

Compile Coderive code to bytecode or native assembly for deployment.

  • Bytecode compilation
  • Native assembly output
  • MTOT optimization
java -cp coderive.jar cod.runner.CompilerRunner --native program.cod

CommandRunner

Unified runner with multiple execution modes in a single command.

  • Interpret or compile
  • Flexible output options
  • All features in one tool
java -cp coderive.jar cod.runner.CommandRunner -c program.cod

CoderiveREPL

Interactive environment for experimenting with Coderive code.

  • Real-time evaluation
  • Multi-line input support
  • State management
java -cp coderive.jar cod.runner.CoderiveREPL

Getting Started

1. Installation

Clone the repository and build the project:

git clone https://github.com/your-username/coderive.git cd coderive ./gradlew build

2. Write Your First Program

Create a simple Coderive program:

output "Hello, Coderive!"

3. Run Your Code

Execute your program with the interpreter:

java -cp build/libs/coderive.jar cod.runner.InterpreterRunner hello.cod
