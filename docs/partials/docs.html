Getting Started

# Download curl -O https://github.com/DanexCodr/Coderive/releases/latest/download/coderive.jar # Create your first program echo 'unit hello share main() { out("Hello, Coderive!") }' > hello.cod # Run it java -jar coderive.jar hello.cod

Quick Syntax Overview

Variables

Type inference, explicit types, and lazy arrays

name := "Coderive" age: int = 19 nums := [1 to 1M]

Functions

Lambda syntax and multiple returns

add := (\a, b) a + b calc := (\a, b) { ~> sum: a + b }

Loops & Ranges

Powerful range syntax with steps

for i in 1 to 10 by 2 { out(i) } slice := arr[0 to 50 by 5]
